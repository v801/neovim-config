# :sparkles: neovim for OSX :sparkles:

#### clone repo
`git clone git://github.com/v801/neovim-config.git ~/.nvim`

#### move into ~/.nvim and run install script
`cd ~/.nvim && install.sh`

#### install plugins via vim-plug
run `nvim` and then run `:PlugInstall`

#### install eslint for javascript linting
`npm i eslint --global`
